## Files in This Repository

### **Program-1.java**
- Prints series based on input value.

### **Program-2.java**
- Calculator program using class.
- Supports:
  - Addition  
  - Subtraction  
  - Multiplication  
  - Division  
  - Modulo  
- Takes 3 inputs: `a`, `b`, and `operation`.

### **Program-3.java**
- Prints odd number series based on rules:
  - If input is odd → print first *n* odd numbers  
  - If input is even → print odd numbers up to nearest smaller odd count

### **Program-4.java**
- Counts how many numbers in an array are divisible by numbers 1 to 9.
- Outputs result in dictionary/map format.
